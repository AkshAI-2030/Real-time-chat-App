# 💬 Real-Time Chat App

A real-time chat application built using Node.js, Express, React, and Socket.io, with MongoDB for data storage.

---

## 🌟 Features
- Real-time messaging with Socket.io
- User authentication (Register/Login)
- Secure password hashing with bcrypt
- Message persistence with MongoDB
- REST API for fetching users and messages

--- 

## 📡 Live Demo URL
[Live Chat App](https://real-time-chat-app-frontend-ashen.vercel.app/)

---

## 🛠️ Tech Stack
- **Frontend:** React
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Real-time Communication:** Socket.io
- **Authentication:** JWT, bcrypt

---

## 🔄 Setup

### Prerequisites
- Node.js installed
- MongoDB database

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/AkshAI-2030/Real-time-chat-App.git
   cd Real-time-chat-App
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Set up environment variables (`.env` file):
   ```sh
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
   ```

4. Start the backend server:
   ```sh
   cd backend
   npm install
   node index.js
   ```

5. Navigate to the frontend directory and start the React app:
   ```sh
   cd frontend
   npm install
   npm start
   ```
